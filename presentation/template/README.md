# 발표용 PPTX 템플릿 (편집 가능)

`타임소프트콘_템플릿.pptx` — PowerPoint에서 **그대로 열어 글자·도형을 직접 수정**할 수 있는
네이티브 템플릿이다. (Marp 빌드물처럼 슬라이드 전체가 이미지로 박히지 않는다.)

## 디자인 톤 (라이트 / 하늘색 — 제품 로그인 화면과 동일한 결)
프로젝트 프론트 메인 컬러(`frontend/src/styles/globals.css`)에서 추출.

| 역할 | HEX | 출처 |
|---|---|---|
| 메인 | `#1B67DA` | `--primary` (216 78% 48%) |
| 딥 | `#0E4EAF` | `--accent-deep` |
| 페이지 배경(옅은 하늘색) | `#F4F8FF`→`#E8F0FF` | 톤 보강 |
| 배경 장식 원 | `#DCE8FF` | 톤 보강 |
| pill 배지 | `#E1ECFF` / 글자 `#1B57C0` | 톤 보강 |

- 전체 밝고 가벼운 톤: **일반 본문 배경도 하늘색 틴트 `#EEF4FF`** (표지/섹션은 화이트→하늘색 그라데이션)
- 제목색: 딥네이비 `#14315F` · 파랑 `#1B67DA`는 액센트(라인·로고·카드 수치)에만
- 표지: pill 배지 + 큰 제목 + 파랑 액센트 라인 + 하단 화이트 지표 카드(라벨/수치/설명)
- **이미지+캡션 레이아웃**: 그림이 자동으로 **흰 둥근 카드** 위에 올라가 틴트 배경에서 이음새 없이 보임
- 좌상단 로고 마크, 하단 헤어라인 푸터, 제목 옆 짧은 파랑 라인
- 한글 폰트 `Malgun Gothic` (윈도우 어디서나 열림)

## 레이아웃(슬라이드 마스터)
홈 ▸ 새 슬라이드 ▸ 아래 레이아웃에서 선택:

| 이름 | 용도 |
|---|---|
| 표지 | 발표 첫 장 (제목/부제) |
| 섹션 구분 | 챕터 전환 (번호+제목) |
| 제목+본문 | 가장 많이 쓰는 글머리 본문 |
| 2단 본문 | 좌우 비교/병렬 |
| 비교 | 좌우 헤더+본문 |
| 제목만 / 빈 화면 | 다이어그램·이미지 자유 배치 |
| 본문+캡션 / **이미지+캡션** | 그림 + 설명 (이미지+캡션은 그림 자리에 사진 넣으면 흰 카드 자동 적용) |

> **틴트 배경에 흰 배경 그림(UML·스크린샷)을 넣을 때**: `이미지+캡션` 레이아웃을 쓰면
> 그림 자리(그림 placeholder)에 사진을 넣는 순간 흰 카드 위에 얹혀 깔끔하다.
> 다른 레이아웃에서 자유 배치할 땐, 그림 뒤에 흰 둥근 사각형(그림자)을 깔고 맨 뒤로 보내면 같은 효과.

## 색/폰트 일괄 변경
보기 ▸ **슬라이드 마스터**에서 한 번 바꾸면 전체 슬라이드에 반영된다.
(제목색·본문 크기·테마색이 마스터에 박혀 있음.)

## 템플릿 재생성 (선택)
색/레이아웃을 코드로 바꾸고 싶으면 생성 스크립트를 수정 후 실행:

```powershell
python presentation/template/build-template.py
```

- 팔레트는 스크립트 상단 `MAIN/DEEP/PAGEBG/PILLBG/CIRCLE` 상수.
- 의존성: `python-pptx` (이미 설치됨).
- 산출물에는 예시 슬라이드 4장(표지/섹션/본문/2단)이 들어 있다 — 보고 나서 지우고 쓰면 된다.
- 표지의 pill 배지·지표 카드는 예시 표지 슬라이드 위에 **편집 가능한 도형**으로 올라가 있다
  (레이아웃이 아니라 슬라이드에 있어 글자·수치를 바로 고칠 수 있음).

## 미리보기 다시 뽑기 (PowerPoint COM, 윈도우)
```powershell
$d="D:\Univercity\2_1SWProject\presentation\template"
$f=(Get-ChildItem (Join-Path $d "*.pptx"))[0].FullName
$ppt=New-Object -ComObject PowerPoint.Application
$p=$ppt.Presentations.Open($f,$true,$false,$false); $i=1
foreach($s in $p.Slides){$s.Export((Join-Path $d ("preview_{0:D2}.png" -f $i)),"PNG",1600,900);$i++}
$p.Close(); $ppt.Quit()
```
