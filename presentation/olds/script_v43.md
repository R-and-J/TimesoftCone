# Part 3 · Q&A (5분, 구두 / 슬라이드 없음)

| 질문 | 30초 답변 |
|---|---|
| "P 단위가 콘으로?" | 학교 프로젝트의 *내부 가상 통화* 정체성. '포인트'는 외부 채널과 헷갈려 **콘**으로 통일. |
| "동시 입찰?" | SQLite 쓰기락 + 단일 트랜잭션. 후행은 fresh-read에서 currentPrice 재확인. |
| "낙찰 시 외부 콜?" | 현재 InternalLeaveAdapter — 로컬 DB만. 외부 그룹웨어 연동은 Outbox 어댑터 교체. |
| "배당 두 번?" | `dividend_run.year` UNIQUE — 시연 4-4에서 보여드림. |
| "스토어 카테고리 추가?" | `/admin/store`에서 행 추가만. 사용자 카탈로그 자동 반영. |
| "회사 데이터 격리?" | JWT companyId 자동 스코프. ADMIN(super)만 스위처. |
| "테스트?" | 도메인 unit `npm test` / E2E `npm run test:e2e`. |
| "감사성?" | 원장도 스토어 상품도 INSERT-only. UPDATE/DELETE 사후 조작 불가. |

부록 슬라이드(A1 기술 스택 · A2 헥사고날 · A3 백업 캡쳐) — 질문 따라 호출.
